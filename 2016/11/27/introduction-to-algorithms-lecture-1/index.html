Is insertion sort fast?

Merge sort

Key subroutine: Merge

两个已经排好序的数组。分别有两个指针指向首元素，比较大小，小的就提取出来，然后指针指向后一位。时间与两个数组之和成正比。

Recursive

T(n) = theta(1), if n = 1 (use omit)
2T(n/2) + theta(n), if n > 1

Recursive tree. T(n) = 2T(n/2) + cn, const c > 0
